# compiler-sfc 示例项目
说明了如何处理 Vue 文件并将其渲染到页面的过程，使用的是 `@vite/compiler-sfc` 这个包

安装依赖
```shell
pnpm i
```
编译 Vue 文件
```shell
# 运行 esno src/build.ts
pnpm run dev
```
本地启动一个 server 并预览打开 `index.html`

VSCode 可以这样快速打开 `html` 页面
